# Anish Weds Shalini

Premium Hindu wedding invitation website — 25 November 2026 · Hotel Vishuwa · Birgunj, Nepal.

## Local preview

```bash
cd anish-shalini-wedding-2026
npm install
npm run dev
```

Then open **http://localhost:5173** in your browser.

If `node` / `npm` are not on your PATH, use the local toolchain already in this project:

```bash
export PATH="$PWD/.tools/node-v22.14.0-darwin-x64/bin:$PATH"
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Content editing

Wedding details, events, navigation, and shlokas live in `src/data/`.
Replace placeholder copy and add photos under `src/assets/`.
